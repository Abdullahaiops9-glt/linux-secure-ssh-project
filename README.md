# 🔐 Linux Multi-User + Secure SSH Project

## 🎯 Overview
This project demonstrates secure Linux system administration using multi-user setup and SSH hardening.

---

🔹 Project : Multi-User System (admin/dev/guest) & Secure SSH Setup
-- Create users & groups
-- Assign roles (admin, dev, guest)
-- Set permissions on folders
-- Setup SSH key login (no password)
-- Disable root login
-- Disable password authentication

## ⚙️ Features
- Multi-user role-based setup
- Secure directory permissions
- SSH key-based authentication
- Root login disabled
- Password authentication disabled

##  🚀 Result
-- Admin full control
-- Dev secure SSH access
-- Guest restricted
-- Root login disabled
